## Prerequisites

- [Zig][] [0.6.0][]

[Zig]:   https://ziglang.org
[0.6.0]: https://ziglang.org/download/#release-0.6.0

## Installation

### Installation from Source Code

```bash
$ git clone --recursive https://github.com/conreality/conreality-sdk.git

$ cd conreality-sdk

$ make

$ sudo make install
```
